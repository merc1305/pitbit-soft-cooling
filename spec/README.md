# Specification Directory

This directory contains the formal documentation for the Pitbit Soft Cooling project, following the Software Design Document (SDD) approach.

## Files

- [SDD.md](SDD.md): The Software Design Document detailing goals, scope, requirements, and "Definition of Done".
- [DECISIONS.md](DECISIONS.md): A log of key architectural decisions, their rationale, and trade-offs.

## Purpose
The documents in this directory serve as the "Source of Truth" for the project's design. Any major changes to the implementation should be preceded or accompanied by updates to these specifications.
